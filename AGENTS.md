# AGENTS.md

Canonical AI agent instructions for this workspace live in
[`CLAUDE.md`](CLAUDE.md). The file applies equally to any AI coding
agent — Claude Code, Cursor, GitHub Copilot, Codex, Aider, OpenAI's
Codex CLI, etc.

Reading order:

1. [`CLAUDE.md`](CLAUDE.md) — workspace-specific conventions, lint
   policy, architectural rules, and the "act as a senior dev partner"
   guidance.
2. [`README.md`](README.md) — what the workspace is, how to build and
   run it, and how it mirrors the structure of `sdh-fleet-client`.
3. The Rust source itself — `src/main.rs`, `contracts/src/lib.rs`,
   `greeter/src/lib.rs`. All three are small (< 100 lines each) and
   are documented inline.
